# 🏨 DataHotel

Aplicación web de gestión hotelera y análisis de indicadores de Revenue Management desarrollada como Proyecto Final del CFGS Desarrollo de Aplicaciones Multiplataforma (DAM).

## 📋 Descripción

DataHotel es una plataforma orientada a pequeños y medianos establecimientos hoteleros que permite centralizar la gestión operativa y visualizar indicadores clave para la toma de decisiones.

La aplicación integra en una única herramienta funcionalidades relacionadas con la gestión de clientes, habitaciones, reservas y análisis de rendimiento hotelero.

## 🚀 Funcionalidades principales

* Gestión de clientes.
* Gestión de habitaciones.
* Gestión de reservas.
* Sincronización simulada de reservas desde distintos canales.
* Dashboard analítico con KPIs hoteleros.
* Indicadores de ocupación, ADR y RevPAR.
* Simulador de tarifas orientado al Revenue Management.
* Consulta del estado operativo del hotel.
* Exportación de información a Excel.

## 📊 Dashboard Revenue

El sistema incorpora un dashboard interactivo que permite visualizar:

* Ocupación.
* Ingresos.
* ADR (Average Daily Rate).
* RevPAR (Revenue per Available Room).
* Cancelaciones.
* Distribución de ingresos por canal.
* Reservas por tipo de habitación.
* Reservas por tipo de canal.
* Ingresos por tipo de canal.

## 🛠️ Tecnologías utilizadas

* Python
* Streamlit
* MySQL
* Pandas
* Plotly
* OpenPyXL

Durante el desarrollo del proyecto se utilizó Docker para facilitar la ejecución y despliegue de la base de datos MySQL en un entorno aislado y reproducible.
